# image_grabber

A lightweight utility designed to automatically scrape, download, and organize bulk images from specified search terms.

---

## 🚀 Features

* **Bulk Downloading:** Fetch multiple images simultaneously based on your target keywords.
* **Automated Organization:** Automatically creates and sorts downloaded images into dedicated folders named after your search terms.
* **Lightweight & Fast:** Built with efficiency in mind, using minimal dependencies for a quick setup.
